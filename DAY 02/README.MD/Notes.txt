

## 🗓️ Day 02 — Worker Demographics and Income 🧑‍💻💰

### 📘 Dataset Overview

The **`workers`** dataset is a classic demographic and socio-economic collection, often derived from census data. Its main goal is to **predict whether an individual’s annual income exceeds $50,000** based on multiple personal and professional attributes. 🎯

---

### 📊 1. Data Structure and Scope

* **Total Records:** `43,957` individual worker records 🤯
* **Primary Goal:** Binary classification using the target column **`income_>50K`**

  * `1` → Income **> $50K** ⬆️
  * `0` → Income **≤ $50K** ⬇️
* **Key Variable Categories:**

  * **Demographics:** `age`, `gender`, `race`, `marital_status`, `native_country` 🌍
  * **Work/Education:** `workclass`, `occupation`, `education`, `educational_num`, `hours_per_week` 📚
  * **Financial/Census:** `fnlwgt`, `capital_gain`, `capital_loss` 💵

---

### 🧠 2. Key Insights from SQL Analysis

The SQL exploration revealed several patterns and trends within the workforce: 🧐

#### 🏢 Work and Occupation

| Finding                         | Details                                                                                                               |
| :------------------------------ | :-------------------------------------------------------------------------------------------------------------------- |
| **Most Common Workclass (Q13)** | `Private` sector dominates with **30,587 workers**.                                                                   |
| **Top 3 Occupations (Q12)**     | `Craft-repair` (5,519) 🛠️, `Prof-specialty` (5,518) 🔬, and `Exec-managerial` (5,506) 👔 are the largest categories. |
| **Average Weekly Hours (Q14)**  | Most groups average ~40 hours/week; however, Black workers average slightly less (≈38.7 hours).                       |

---

#### 🎂 Age Distribution (Q7 & Q19)

* **Average Age:** ~**38.6 years**
* **Largest Age Group:** **31–50 (“Middle”)**, with **20,844 workers** 💪

This indicates a predominantly mid-career workforce demographic.

---

#### 🎓 Income and Education (Q17 & Q20)

There’s a clear correlation between **education level** and **likelihood of earning over $50K**:

| Education Level | % Earning > $50K |
| :-------------- | ---------------: |
| **Prof-school** |    **73.66%** 🥇 |
| **Doctorate**   |    **71.64%** 🚀 |
| **Masters**     |       **55.52%** |
| **Bachelors**   |       **41.40%** |

Higher education strongly boosts income potential — showing that **learning pays off literally** 💰📈
